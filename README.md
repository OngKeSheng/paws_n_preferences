# Paws & Preferences

A cat swiping app where you swipe through cat pictures and find out which ones you like. Like Tinder but for cats.

## What it does

Swipe right if you like a cat, swipe left if you don't. At the end, you'll see all the cats you liked.

## How to use

**On phone:** Swipe the card left or right, or use the buttons.

**On desktop:** Click and drag the card, or use the buttons. You can also use arrow keys.

## Running it locally

Just click [here](https://ongkesheng.github.io/paws_n_preferences/) in your browser. That's it.

Done!
